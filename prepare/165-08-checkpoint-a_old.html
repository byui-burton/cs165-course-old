08 Prepare : Checkpoint A - Inheritance

After completing (or while completing) the preparation material for this week, complete the following exercise.

Overview

This checkpoint is intended to help you practice the syntax of basic inheritance.

Your roommate is always forgetting where they saved their mp3s and video files on their computer and asked you to write a program to keep track of them.

You realize that because these two types of media files are different you'll want a class for a AudioFile and one for Video, but as you start to design them, you realize that they have a lot of things in common.

In particular, the following are needed in both classes:

  • filename : string
  • title : string
  • length : int

The following are unique to an AudioFile:

  • artist : string
  • album : string

The following are unique to a Video:

  • author : string

Instructions

  1. mkdir check07a cd check07a cp /home/cs165new/check07a/* .
  2. This class has the following class diagram:
    3. IceCream
    -flavor : string
    -price : float
    +IceCream()
    +IceCream(flavor, price)
    +getTotalPrice() : float
    +prompt() : void
    +display() : void
  3. Add to this class a public static variable named salesTax.
  4. Give this variable a default value of 0.06.
  5. Change the getTotalPrice() function to include the tax in the price.
  6. A main function is provided for you in check07a.cpp. It declares two IceCream objects, displays them, asks for an alternate sales tax, and displays them again. Currently (because you hadn't written it yet), it does not actually change the sales tax value.
  7. Change main, where noted in the comments, to actually set the new sales tax value.
