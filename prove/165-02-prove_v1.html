02 Prove : Assignment - Digital Forensics

Overview

ACME Widget Corporation just experienced their first digital security breach. One of their secret widget designs just appeared on the Internet. Now they've hired you as a digital forensics investigator to help determine what additional data might have been involved in the breach.

One of the first steps in digital forensics is to inspect system logs, particularly file access and user account logs. To assist you in this investigation, you've been given a log from ACME's network server that contains a list of file access times and the users that accessed them.

Write a program to scan through that log to identify users who accessed files in a particular window of time.

Instructions

Write a C++ program according to the following:

  1. Create a struct for a Date that holds a year, month, and day.
  2. Create a struct for an Access Record that contains a username, a filename, and an access Date (i.e., a Date struct).
  3. Prompt the user for the name of a file.
  4. Read in a list of Access Records from the file and store it in an array of structs.
  5. Prompt the user for a start date and an end date for consideration.
  6. Display a list of all files that were accessed during that period along with the users that accessed them.
  7. For this assignment you may assume that the information entered is all valid
  8. Don't forget to break your program into meaningful, cohesive functions!

A few hints

  • A common technique for comparing custom items is to create a function, such as compareDate, that takes two Date structs and returns -1 if the first one comes before the second, 0 if they are equal, and 1 if the first comes after the second.
  • It is also common to create a function, such as displayRecord, that would take a struct and display it in a proper format.
  • Get started early, there are a few tricky aspects to this assignment.
